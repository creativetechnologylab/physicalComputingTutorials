# What is the OLED monochrome display?
The OLED monochrome display is a small (tiny) and high-readability display. It is useful for displaying data, e.g. weather information or small graphics like what you see on Tamagotchi. For more information, please visit [here](https://learn.adafruit.com/monochrome-oled-breakouts/overview).

![image](https://github.com/creativetechnologylab/physicalComputingTutorials/assets/64136454/877912b3-4043-4c00-9074-b61f1aa2a91a)


In this tutorial, we will be using Adafruit SSD1306 128 x 64 OLED with I2C communication. It can support SPI communication as well. 

<p class="callout warning"><b>Display Configuration</b><br/>
If you have the older non-STEMMA version of the OLED, you'll need to solder the two jumpers on the back of the OLED. Both must be soldered 'closed' for I2C to work! </p>

![image](https://github.com/creativetechnologylab/physicalComputingTutorials/assets/64136454/4ab5d3f1-f5a5-47be-a4f3-5dfe5cd90ffc)


# Wiring
Wiring up the sensor is simple:

1. Power (VIN to 5V)
1. Ground (GND to GND)
1. Data to Arduino SDA pin (A5 on Uno)
1. CLK to Arduino SCL pin (A4 on Uno)

# Library
To use this code you will need the [Adafruit_SSD1306 Library](https://github.com/adafruit/Adafruit_SSD1306).
We have a tutorial on [how to install a library](https://github.com/creativetechnologylab/physicalComputingTutorials/blob/main/How%20to%20install%20libraries.md) here.
